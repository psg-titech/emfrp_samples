# SimFanController
A simple controller for electric fan (simulator)  
This example is intended to run on a desktop (e.g., Linux, Mac)

## To build & run

With PlatformIO
```sh
$ (cd src && make)
$ platformio run
$ .pioenvs/native/program
```

Without PlatformIO
```sh
$ cd src
$ make exe
$ ./SimFanController
```
